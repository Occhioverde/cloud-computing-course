# Well done!

You have completed the Kubernetes "Zero to Hero" deployment exercise. Here is what you practiced:

**Step 1** — Bootstrapped a single-node Kubernetes cluster from scratch using `kubeadm`, configured `kubectl`, and installed the Flannel CNI.

**Step 2** — Created your first Pod running MongoDB, learning how to define containers and labels in a YAML manifest.

**Step 3** — Exposed the MongoDB Pod internally using a `ClusterIP` Service, enabling stable network discovery within the cluster.

**Step 4** — Deployed a web-based administration UI (Mongo Express) and exposed it to the outside world using a `NodePort` Service.

**Step 5** — Applied your database skills by using the Mongo Express web interface to create databases, insert nested documents, and execute complex JSON queries.
